# Keeping Wisdom (守智慧)

A mobile application for recording, transcribing, and summarizing audio conversations.

## Features

- Audio recording with high quality settings
- Automatic transcription and note-taking
- Key point summarization
- Bilingual support (Traditional Chinese and English)
- Adjustable text sizes for accessibility
- Playback controls with variable speed

## Tech Stack

- React Native with Expo
- TypeScript
- Zustand for state management
- Expo AV for audio recording and playback
- Expo Router for navigation
